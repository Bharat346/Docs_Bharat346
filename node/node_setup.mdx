
## Getting Started with Node.js

Node.js is a powerful JavaScript runtime built on Chrome's V8 engine. Setting it up correctly is crucial for development.



## Installation Steps

To start using Node.js, follow these steps:

1. **Download Node.js**: Visit the [official Node.js website](https://nodejs.org/) and download the latest LTS version.
2. **Install Node.js**: Run the installer and follow the setup instructions.
3. **Verify Installation**: Open a terminal and type:

   ```bash
   node -v
   npm -v
   ```
   
   This will display the installed versions of Node.js and npm.

## Common Node.js Commands

Here are some common commands you'll frequently use:

| Command             | Description                                     |
|--------------------|-------------------------------------------------|
| `node -v`         | Check the installed Node.js version             |
| `npm init -y`     | Initialize a new Node.js project                |
| `npm install`     | Install dependencies from package.json          |
| `node app.js`     | Run a Node.js script                            |


## Setting Up a Basic Server

A simple way to create a Node.js server using Express:

```javascript
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello, Node.js!');
});

app.listen(3000, () => {
  console.log('Server running on port 3000');
});
```

Run the script with:
```bash
node server.js
```
